## 1. Homepage Tour

### Overview

The HausFinder homepage serves as the central hub for property discovery in Honiara, Solomon Islands. It features a dramatic cinematic design with a deep teal and burnt orange gradient backdrop.

### Visual Elements

| Element | Location | Purpose |
|---------|----------|---------|
| **HausFinder Logo** | Top-left corner | Brand identity with home icon |
| **Navigation Menu** | Top center | Home, Search, Map, Favorites, Dashboard |
| **Theme Toggle** | Top-right | Switch between dark and light themes |
| **List Property Button** | Top-right (orange) | Quick access to create listings |
| **Hero Section** | Center | Main headline with search bar |
| **Quick Filter Buttons** | Below search | One-click property type filters |
| **Featured Properties** | Below hero | Grid of highlighted listings |

### Screenshot Description

**Scene 1: Homepage Hero (Dark Theme)**

The homepage displays a dramatic dark gradient background blending deep teal (#0f172a) with burnt orange accents. The main headline reads "Find Your Perfect Home in Honiara" with "Home" highlighted in orange (#ea580c). A prominent search bar sits below with an orange "Search" button. Four quick filter buttons appear beneath: "Houses for Sale," "Apartments for Rent," "All Rentals," and "All Sales."

### Suggested Narration

> "Welcome to HausFinder, the premier property platform for the Solomon Islands. The homepage greets you with a stunning cinematic design. Notice the prominent search bar at the center—this is your starting point for finding properties. Below, you'll see quick filter buttons that let you jump directly to specific property types. Scroll down to discover featured properties handpicked from prime Honiara locations."

### Key Actions to Demonstrate

1. Point out the navigation menu items
2. Highlight the search bar functionality
3. Click each quick filter button to show instant filtering
4. Scroll down to reveal featured property cards
5. Hover over a property card to show the favorite heart icon

## 3. Using Advanced Filters

### Overview

Advanced filters allow users to narrow down property searches by multiple criteria simultaneously.

### Visual Elements

| Filter | Options | Default |
|--------|---------|---------|
| **Status** | All, For Rent, For Sale | All |
| **Property Type** | All Types, House, Apartment, Land, Commercial | All Types |
| **Min Price** | Numeric input (SBD) | 0 |
| **Max Price** | Numeric input (SBD) | Any |
| **Bedrooms** | Any, 1+, 2+, 3+, 4+, 5+ | Any |

### Screenshot Description

**Scene 3: Advanced Filters Panel**

When the filter icon is clicked, a filters panel expands below the search bar. Five filter controls appear in a horizontal row: Status dropdown (showing "All"), Property Type dropdown (showing "All Types"), Min Price input field, Max Price input field, and Bedrooms dropdown (showing "Any"). The filters panel has a subtle background that distinguishes it from the main content.

### Suggested Narration

> "Click the filter icon to reveal powerful search filters. You can filter by status—whether you're looking to rent or buy. Choose a property type like House, Apartment, Land, or Commercial. Set your budget with minimum and maximum price fields—remember, all prices are in Solomon Islands Dollars. Finally, specify the minimum number of bedrooms you need. The results update automatically as you adjust these filters."

### Key Actions to Demonstrate

1. Click the filter icon to expand filters
2. Select "For Sale" from Status dropdown
3. Select "House" from Property Type
4. Enter "1000000" in Min Price
5. Show filtered results (houses for sale over 1M SBD)
6. Reset filters to show all properties

## Video Production Notes

### Recommended Video Structure

| Tutorial | Duration | Sections Covered |
|----------|----------|------------------|
| **Getting Started** | 3-5 min | Sections 1, 12 |
| **Finding Properties** | 5-7 min | Sections 2, 3, 4, 5 |
| **Listing Your Property** | 7-10 min | Sections 6, 7, 8 |
| **Managing Your Account** | 4-6 min | Sections 9, 10, 11 |

### Recording Tips

1. **Screen Resolution**: Record at 1920x1080 for optimal quality
2. **Mouse Movements**: Move the cursor slowly and deliberately
3. **Pauses**: Pause briefly on important elements before clicking
4. **Zoom**: Use screen zoom for small UI elements
5. **Annotations**: Add arrows or highlights in post-production

### Accessibility Considerations

1. Include closed captions for all narration
2. Describe visual elements verbally for screen reader users
3. Use high contrast when highlighting elements
4. Provide text alternatives for all screenshots
