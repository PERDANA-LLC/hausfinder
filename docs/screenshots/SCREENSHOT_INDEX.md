## Recording Notes

### Recommended Screen Resolution
- **Desktop:** 1920x1080 (Full HD)
- **Mobile Demo:** 390x844 (iPhone 14 Pro)

### Browser Setup
- Use Chrome/Chromium for consistency
- Clear browser cache before recording
- Disable browser extensions
- Use incognito mode for clean state

### Recording Tips
1. Record at 60fps for smooth playback
2. Use consistent mouse movement speed
3. Pause 2-3 seconds on key screens
4. Add zoom effects on important UI elements
5. Include cursor highlights for clicks
