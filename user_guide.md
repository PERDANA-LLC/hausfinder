## Introduction

HausFinder is the premier property listing platform designed specifically for the Solomon Islands market, with a focus on Honiara. Whether you are looking to rent an apartment, buy a family home, invest in land, or list your own property, HausFinder provides all the tools you need in one place.

### Key Features

| Feature | Description |
|---------|-------------|
| Property Search | Find properties by location, type, price, and amenities |
| Interactive Map | Browse properties visually on a Google Map of Honiara |
| AI-Powered Descriptions | Generate compelling property descriptions automatically |
| Multi-Image Uploads | Showcase properties with multiple high-quality photos |
| Favorites System | Save properties you love for easy access later |
| Contact Forms | Reach property owners directly through the platform |
| User Dashboard | Manage all your listings and inquiries in one place |
| Admin Dashboard | Full user management and role control for administrators |

### Supported Property Types

HausFinder supports four main property categories:

- **Houses** – Single-family homes, villas, and residential properties
- **Apartments** – Units, flats, and multi-family dwellings
- **Land** – Vacant lots, development parcels, and beachfront land
- **Commercial** – Office buildings, retail spaces, and business properties

## Listing a Property

### Creating a New Listing

To list a property on HausFinder:

1. Click **List Property** in the navigation menu (requires login)
2. Complete the listing form with all required information

### Listing Form Fields

#### Basic Information

| Field | Description | Required |
|-------|-------------|----------|
| **Title** | A compelling headline for your property | Yes |
| **Property Type** | House, Apartment, Land, or Commercial | Yes |
| **Listing Type** | For Rent or For Sale | Yes |
| **Price** | Amount in SBD | Yes |

#### Property Details

| Field | Description | Required |
|-------|-------------|----------|
| **Bedrooms** | Number of bedrooms | No |
| **Bathrooms** | Number of bathrooms | No |
| **Area** | Size in square meters | No |
| **Amenities** | Features like pool, parking, AC, etc. | No |

#### Location

| Field | Description | Required |
|-------|-------------|----------|
| **Location** | Neighborhood or area name | Yes |
| **Address** | Full street address | No |
| **Map Coordinates** | Latitude and longitude for map display | No |

#### Description

You can write your own description or use the **AI-powered description generator**:

1. Fill in the basic property details first
2. Click **Generate with AI** button
3. Review and edit the generated description as needed
4. The AI uses your property details to create compelling marketing copy

#### Photos

Upload multiple images to showcase your property:

1. Click the upload area or drag and drop images
2. Supported formats: JPG, PNG, WebP
3. First image becomes the primary/featured image
4. Reorder images by drag and drop
5. Remove images by clicking the X button

### Publishing Your Listing

1. Review all entered information
2. Click **Create Listing** to publish
3. Your listing will appear immediately on the platform

## Admin Dashboard

The Admin Dashboard provides powerful management capabilities for platform administrators. This section is only accessible to users with **Admin** or **Super Admin** roles.

### Accessing the Admin Dashboard

Administrators can access the dashboard through a dedicated login:

1. Navigate to `/admin-login` in your browser
2. Enter your admin email address
3. Enter your admin password
4. Click **Sign In to Admin**
5. Upon successful authentication, you will be redirected to the Admin Dashboard

### User Roles

HausFinder supports four user roles with different permission levels:

| Role | Description | Capabilities |
|------|-------------|-------------|
| **User** | Standard platform user | Browse, search, favorite properties, contact owners |
| **Agent** | Real estate agent | All User capabilities plus enhanced listing features |
| **Admin** | Platform administrator | All Agent capabilities plus user management |
| **Super Admin** | System owner | Full unrestricted access, immutable account |

### Dashboard Overview

The Admin Dashboard displays key metrics at a glance:

- **Total Users** – Complete count of registered users
- **Admins** – Number of users with Admin or Super Admin roles
- **Agents** – Number of registered real estate agents
- **Regular Users** – Standard user accounts

### User Management

Administrators can perform full CRUD (Create, Read, Update, Delete) operations on user accounts.

#### Creating a New User

1. Click the **Add User** button
2. Fill in the required fields:
   - **Name** – User's full name (required)
   - **Email** – Valid email address (required)
   - **Password** – Optional password for admin login capability
   - **Role** – Select from User, Agent, Admin, or Super Admin
   - **Phone** – Optional contact number
3. Click **Create User** to save

#### Editing a User

1. Locate the user in the User Management table
2. Click the **Edit** (pencil) icon in the Actions column
3. Modify any fields as needed
4. Click **Update User** to save changes

#### Changing User Roles

To change a user's role:

1. Click the Edit icon for the target user
2. Select the new role from the Role dropdown
3. Save changes

**Note:** The Super Admin account (`superadmin@guest.com`) is immutable and cannot have its role changed.

#### Deleting a User

1. Click the **Delete** (trash) icon for the user
2. Confirm the deletion in the dialog
3. The user account will be permanently removed

**Note:** Immutable accounts (Super Admin) cannot be deleted.

### Super Admin Account

The platform includes a built-in Super Admin account with special protections:

| Property | Value |
|----------|-------|
| **Email** | superadmin@guest.com |
| **Password** | guest.com@superadmin |
| **Role** | Super Admin |
| **Status** | Immutable (cannot be deleted or demoted) |

The Super Admin account:

- Has unrestricted access to all platform features
- Cannot be deleted by any user
- Cannot have its role changed
- Is automatically created on system initialization
- Is indicated by a lock icon in the user table

### Security Considerations

- Admin login uses secure password hashing (bcrypt)
- Session tokens are JWT-based with 7-day expiration
- All admin actions are authenticated server-side
- The admin login page is separate from regular user authentication

### Admin Navigation

The Admin Dashboard header provides quick navigation:

| Button | Function |
|--------|----------|
| **Back to Site** | Return to the main HausFinder platform |
| **Logout** | End admin session and return to login page |

## FAQ

### General Questions

**Q: Is HausFinder free to use?**

A: Yes, browsing properties and creating an account is completely free. Listing properties is also free during the platform launch period.

**Q: What currency are prices displayed in?**

A: All prices are displayed in Solomon Islands Dollars (SBD).

**Q: Can I list properties outside of Honiara?**

A: Currently, HausFinder focuses on the Honiara area, but we plan to expand to other regions of the Solomon Islands in the future.

### Account Questions

**Q: How do I reset my password?**

A: Use the password reset option on the login page, or contact support for assistance.

**Q: Can I have multiple accounts?**

A: We recommend using a single account to manage all your listings and favorites.

### Listing Questions

**Q: How long does my listing stay active?**

A: Listings remain active until you deactivate them or mark the property as sold/rented.

**Q: Can I edit my listing after publishing?**

A: Yes, you can edit any aspect of your listing at any time from your dashboard.

**Q: How many photos can I upload?**

A: You can upload multiple photos per listing. We recommend 5-10 high-quality images for best results.

### Technical Questions

**Q: Why isn't the map loading?**

A: Ensure you have a stable internet connection. The map requires Google Maps to load, which may take a moment on slower connections.

**Q: My images aren't uploading. What should I do?**

A: Check that your images are in a supported format (JPG, PNG, WebP) and not too large. Try compressing large images before uploading.
